# User Manual

## Admin

- Open the admin dashboard after login.
- Review incidents, visitor logs, and emergency alerts.
- Assign security personnel, generate reports, and monitor analytics.

## Security Personnel

- Review assigned incidents and patrol tasks.
- Update incident status and acknowledge emergency alerts.
- Submit patrol reports and communicate with the admin team.

## Staff

- Submit new incident reports using the incident form.
- Track progress and receive safety announcements.
- Use emergency contact details when an urgent issue occurs.

## Visitor / User

- Register an account and complete the verification flow.
- Use the login page to access the visitor dashboard.
- Check visit status, QR pass details, and host approval updates.
