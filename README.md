# Intelligent Security Management System

This repository is being scaffolded as a full-stack safety and security management platform for LASUSTECH final-year project delivery.

## Planned Structure

- `apps/web` - Next.js frontend, landing page, auth pages, and role dashboards
- `apps/api` - Express API, realtime notifications, auth, incidents, visitors, analytics, and reports
- `packages/shared` - shared constants, types, and validation helpers
- `prisma` - PostgreSQL schema, migrations, and seeds
- `docs` - architecture, ERD, API, deployment, and user documentation

## Current Status

The first scaffold includes the web application shell, a responsive landing page, auth entry points, and dashboard routing. The API and database layers will follow in the next step.
